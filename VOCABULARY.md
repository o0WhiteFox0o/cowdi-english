# 📚 Cowdi English — Vocabulary System

## Tổng quan

Hệ thống từ vựng Cowdi English được tổ chức theo **Mind Map** — chia từ vựng thành các **chủ đề (Topic)** → **chủ đề con (Subtopic)** → **từ vựng (Word)**.

- **Mục tiêu:** 3,000 từ
- **Hiện tại:** 702 từ (theo chủ đề) + 210 từ (bài học) = **912 từ**
- **Số chủ đề:** 13
- **Số chủ đề con:** 42
- **File dữ liệu:** `src/data/vocab-topics.js`

## 🛠️ Cấu trúc dữ liệu

Mỗi từ có cấu trúc:

```js
{
  word: 'Apple',           // Từ tiếng Anh
  phonetic: '/ˈæpəl/',    // Phiên âm IPA
  meaning: 'Quả táo',     // Nghĩa tiếng Việt
  example: 'I eat an apple every day.',  // Câu ví dụ
  illustration: '🍎'       // Emoji minh họa
}
```

## 🎮 Tính năng UI

- **Mind Map View:** Hiển thị các chủ đề dạng nút kết nối từ trung tâm
- **Subtopic Drill-down:** Click chủ đề → xem các chủ đề con dạng lưới
- **Flashcard Mode:** Lật thẻ với emoji, phiên âm, phát âm TTS
- **List Mode:** Danh sách từ với trạng thái đã thuộc/đang học
- **Progress Tracking:** Thanh tiến trình cho mỗi topic/subtopic
- **SRS Integration:** Tích hợp thuật toán SM-2 để ôn từ vựng
